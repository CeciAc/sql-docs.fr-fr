# <a name="azdata-bdc-status"></a>État du contrôleur de domaine azdata

[!INCLUDE[tsql-appliesto-ssver15-xxxx-xxxx-xxx](../includes/tsql-appliesto-ssver15-xxxx-xxxx-xxx.md)]

L’article suivant fournit des informations de référence sur les commandes d' **État BDC** dans l’outil **azdata** . Pour plus d’informations sur les autres commandes **azdata** , consultez [référence azdata](reference-azdata.md).

## <a name="commands"></a>Commandes
|     |     |
| --- | --- |
[affichage de l’état du contrôleur de domaine azdata](#azdata-bdc-status-show) | Affiche l’état du cluster Big Data.
## <a name="azdata-bdc-status-show"></a>affichage de l’état du contrôleur de domaine azdata
Affiche l’état du cluster Big Data.
```bash
azdata bdc status show 
```
### <a name="examples"></a>Exemples
État du BDC où l’utilisateur est connecté.
```bash
azdata bdc status show
```
### <a name="global-arguments"></a>Arguments globaux
#### `--debug`
Augmente le détail de la journalisation pour afficher tous les journaux d’activité de débogage.
#### `--help -h`
Affiche ce message d’aide et quitte.
#### `--output -o`
Format de sortie.  Valeurs autorisées: JSON, jsonc, table, TSV.  Valeur par défaut: JSON.
#### `--query -q`
Chaîne de requête JMESPath. Pour [http://jmespath.org/](http://jmespath.org/]) plus d’informations et d’exemples, consultez.
#### `--verbose`
Augmente le détail de la journalisation. Utilisez --debug pour les journaux d’activité de débogage complets.
