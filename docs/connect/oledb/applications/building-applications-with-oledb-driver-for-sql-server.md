# <a name="building-applications-with-ole-db-driver-for-sql-server"></a>Génération d’applications avec OLE DB Driver pour SQL Server
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  Lors du développement d’une application qui utilise le pilote OLE DB pour la bibliothèque SQL Server, il existe un certain nombre de problèmes qui entrent en jeux. Les rubriques de cette section traitent de la plupart de ces problèmes, notamment la mise à niveau de MDAC vers OLE DB pilote pour SQL Server, l’utilisation du pilote OLE DB pour les fichiers d’en-tête et de bibliothèque SQL Server, ainsi qu’une vue d’ensemble des différentes chaînes de connexion qui peuvent être utilisées avec OLE DB pilote pour S Serveur QL Server.

 [Composants OLE DB Driver pour SQL Server](../../oledb/applications/components-of-oledb-driver-for-sql-server.md)  
 Décrit les composants qui composent OLE DB pilote pour SQL Server y compris les fichiers de bibliothèque, de ressources, d’aide et d’en-tête.

 Aborde les différents types de chaînes de connexion qui peuvent être utilisés lors de la connexion à une base de données via OLE DB Driver pour SQL Server.
