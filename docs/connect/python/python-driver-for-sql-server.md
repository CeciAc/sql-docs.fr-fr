# <a name="python-sql-driver"></a>Pilote SQL python

![Télécharger-DownArrow-cercled](../../ssdt/media/download.png)[pour installer le pilote SQL pour Python](../sql-connection-libraries.md#anchor-20-drivers-relational-access)

Vous pouvez vous connecter à un SQL Database à l’aide de Python sur Windows, Linux ou Mac.   
  
## <a name="getting-started"></a>Mise en route  
Plusieurs pilotes SQL Python sont disponibles. Toutefois, Microsoft met ses efforts de test et sa confiance dans le pilote **pyodbc** . Choisissez un pilote et configurez votre environnement de développement en conséquence:
* [Pilote SQL Python – pyodbc](pyodbc/python-sql-driver-pyodbc.md)
* [Pilote SQL Python – pymssql](pymssql/python-sql-driver-pymssql.md)
