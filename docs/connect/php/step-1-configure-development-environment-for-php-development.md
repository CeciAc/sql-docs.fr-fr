# <a name="step-1-configure-environment-for-php-development"></a>Étape 1: configurer l’environnement pour le développement PHP
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]




* Identifiez la version du pilote PHP que vous allez utiliser, en fonction de votre environnement, comme indiqué ici: [Configuration système requise pour les pilotes Microsoft pour PHP pour SQL Server](../../connect/php/system-requirements-for-the-php-sql-driver.md)
* Téléchargez et installez le pilote PHP applicable ici: [Télécharger le pilote Microsoft php](https://www.microsoft.com/download/details.aspx?id=20098)  
* Téléchargez et installez le pilote ODBC applicable ici: [Télécharger le pilote ODBC pour SQL Server](../../connect/odbc/download-odbc-driver-for-sql-server.md)  
* Configurez le pilote PHP et le serveur Web pour votre système d’exploitation spécifique:

### <a name="windows"></a>Windows  
  

* Configurez le chargement du pilote PHP, comme indiqué ici: [chargement des pilotes Microsoft pour PHP pour SQL Server](../../connect/php/loading-the-php-sql-driver.md) 
* Configurez IIS pour héberger des applications PHP, comme indiqué ici: [configuration d’IIS pour les pilotes Microsoft pour PHP pour SQL Server](../../connect/php/configuring-iis-for-php-sql-driver.md)

### <a name="linux-and-mac"></a>Linux et Mac


*   Configurez le chargement du pilote PHP et configurez votre serveur Web pour héberger des applications PHP, comme indiqué ici: [didacticiel d’installation des pilotes php Linux et Mac](../../connect/php/installation-tutorial-linux-mac.md)
