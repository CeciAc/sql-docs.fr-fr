# <a name="multiple-active-result-sets-mars"></a>MARS (Multiple Active Result Sets)

![Download-DownArrow-Circled](../../../ssdt/media/download.png)[Télécharger ADO.NET](../../sql-connection-libraries.md#anchor-20-drivers-relational-access)

MARS (Multiple Active Result Sets) est une fonctionnalité qui permet l’exécution de plusieurs lots sur une seule connexion. Dans les versions précédentes, un seul lot pouvait être exécuté à la fois sur une seule connexion. L’exécution de plusieurs lots avec MARS n’implique pas l’exécution simultanée d’opérations.
