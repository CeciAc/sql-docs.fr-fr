# <a name="getnstring-method-sqlservercallablestatement"></a>Méthode getNString (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère la valeur du paramètre **nchar**, **nvarchar**ou **LONGNVARCHAR** désigné sous la forme d’une chaîne dans le langage de programmation Java.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Créer une vue d’abonnement|Description|  
|----------|-----------------|  
|[Méthode &#40;getNString int&#41;](../../../connect/jdbc/reference/getnstring-method-int.md)|Récupère la valeur du paramètre **nchar**, **nvarchar**ou **LONGNVARCHAR** désigné sous la forme d’une chaîne dans le langage de programmation Java.|  
|[getNString, méthode &#40;java.lang.String&#41;](../../../connect/jdbc/reference/getnstring-method-java-lang-string.md)|Récupère la valeur du paramètre **nchar**, **nvarchar**ou **LONGNVARCHAR** désigné sous la forme d’une chaîne dans le langage de programmation Java.|
