# <a name="isstatementpoolingenabled-method-sqlserverconnection"></a>isStatementPoolingEnabled, méthode (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

 Retourne une valeur indiquant si le regroupement d’instructions est activé ou non pour cette connexion.

## <a name="syntax"></a>Syntaxe  
  
```  
  
public boolean isStatementPoolingEnabled()  
```  

## <a name="return-value"></a>Valeur retournée
 Un **booléenne** qui contient l’indicateur indiquant si le regroupement d’instructions est activé ou non.

## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
 
## <a name="remarks"></a>Notes  
 Cette méthode est disponible à partir de la version du pilote JDBC 6.4 et ultérieur.
