# <a name="getsavepointid-method-sqlserversavepoint"></a>Méthode getSavepointId (SQLServerSavepoint)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Obtient l'ID du point de sauvegarde.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public int getSavepointId()  
```  
  
## <a name="return-value"></a>Valeur retournée  
 Un **int** valeur.  
  
## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notes   
 Cette méthode getSavepointId est spécifiée par la méthode getSavepointId dans l’interface java.sql.Savepoint.
