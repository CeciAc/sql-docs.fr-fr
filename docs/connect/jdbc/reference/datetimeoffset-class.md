# <a name="datetimeoffset-class"></a>Classe DateTimeOffset
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Ce type représente un objet de type Microsoft. Sql. DateTimeOffset et est le type Java qui correspond au type [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] de données **DateTimeOffset** .  
  
 **Package :** microsoft.sql  
  
 **Étend :** java.lang.Object  
  
 **Implémente :** java.lang.Comparable\<DateTimeOffset>, java.io.Serializable  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public final class DateTimeOffset  
```
