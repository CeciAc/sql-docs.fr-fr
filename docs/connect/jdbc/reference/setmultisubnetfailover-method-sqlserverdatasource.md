# <a name="setmultisubnetfailover-method-sqlserverdatasource"></a>Méthode setMultiSubnetFailover (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Définit la valeur de la **multiSubnetFailover** propriété de connexion.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
public void setMultiSubnetFailover(boolean multiSubnetFailover);  
```  
  
#### <a name="parameters"></a>Paramètres  
 *multiSubnetFailover*  
  
 La nouvelle valeur de la **multiSubnetFailover** propriété de connexion.  
  
## <a name="remarks"></a>Notes   
 Pour plus d’informations sur la **multiSubnetFailover** propriété de connexion, consultez [définissant les propriétés de connexion](../../../connect/jdbc/setting-the-connection-properties.md).
