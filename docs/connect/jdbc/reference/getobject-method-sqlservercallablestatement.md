# <a name="getobject-method-sqlservercallablestatement"></a>Méthode getObject (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère la valeur du paramètre désigné en tant qu'objet dans le langage de programmation Java.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Créer une vue d’abonnement|Description|  
|----------|-----------------|  
|[getObject (int)](../../../connect/jdbc/reference/getobject-method-int.md)|Récupère la valeur du paramètre désigné en tant qu'objet dans le langage de programmation Java en fonction de l'index de paramètre.|  
|[getObject (int, java.util.Map)](../../../connect/jdbc/reference/getobject-method-int-java-util-map.md)|Récupère la valeur du paramètre désigné en tant qu’objet dans le langage de programmation Java en fonction de l’index de paramètre fourni, en utilisant l’objet Map spécifié.|  
|[getObject (java.lang.String)](../../../connect/jdbc/reference/getobject-method-java-lang-string.md)|Récupère la valeur du paramètre désigné en tant qu'objet dans le langage de programmation Java en fonction du nom du paramètre.|  
|[getObject (java.lang.String, java.util.Map)](../../../connect/jdbc/reference/getobject-method-java-lang-string-java-util-map.md)|Récupère la valeur du paramètre désigné sous forme d’objet dans le langage de programmation Java en fonction du nom de paramètre fourni, avec l’objet Map donné.|
