# <a name="clearparameters-method-sqlserverpreparedstatement"></a>clearParameters, méthode (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Efface immédiatement les valeurs des paramètres actuels.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public final void clearParameters()  
```  
  
## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notes  
 Cette méthode clearParameters est spécifiée par la méthode clearParameters dans l’interface java.sql.PreparedStatement.
