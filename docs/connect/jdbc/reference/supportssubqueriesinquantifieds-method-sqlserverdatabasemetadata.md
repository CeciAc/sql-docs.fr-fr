# <a name="supportssubqueriesinquantifieds-method-sqlserverdatabasemetadata"></a>Méthode supportsSubqueriesInQuantifieds (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère les informations déterminant si cette base de données prend en charge les sous-requêtes dans les expressions quantifiées.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public boolean supportsSubqueriesInQuantifieds()  
```  
  
## <a name="return-value"></a>Valeur retournée  
 **true** si pris en charge. Dans le cas contraire, la valeur est **false**.  
  
## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notes   
 Cette méthode supportsSubqueriesInQuantifieds est spécifiée par la méthode supportsSubqueriesInQuantifieds dans l’interface java.sql.DatabaseMetaData.
