# <a name="getinstancename-method-sqlserverdatasource"></a>Méthode getInstanceName (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Retourne le nom de l’instance de [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public java.lang.String getInstanceName()  
```  
  
## <a name="return-value"></a>Valeur retournée  
 Un **chaîne** qui contient l’instance de nom, ou null si aucune valeur n’est définie.
