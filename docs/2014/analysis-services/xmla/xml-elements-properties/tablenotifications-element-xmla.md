# <a name="tablenotifications-element-xmla"></a>Élément TableNotifications (XMLA)
  Contient une collection d'éléments [TableNotification](tablenotification-element-xmla.md) utilisée par la commande [NotifyTableChange](../xml-elements-commands/notifytablechange-element-xmla.md) .  
  
## <a name="syntax"></a>Syntaxe  
  
```xml  
  
<NotifyTableChange >  
   ...  
   <TableNotifications>  
      <TableNotification>...</TableNotification>  
   </TableNotifications>  
   ...  
</NotifyTableChange>  
```  
  
## <a name="element-characteristics"></a>Caractéristiques de l'élément  
  
|Caractéristique|Description|  
|--------------------|-----------------|  
|Type de données et longueur|None|  
|Valeur par défaut|None|  
|Cardinalité|0-1 : élément facultatif qui peut apparaître une fois et une seule.|  
  
## <a name="element-relationships"></a>Relations entre les éléments  
  
|Relation|Élément|  
|------------------|-------------|  
|Éléments parents|[NotifyTableChange](../xml-elements-commands/notifytablechange-element-xmla.md)|  
|Éléments enfants|[TableNotification](tablenotification-element-xmla.md)|
