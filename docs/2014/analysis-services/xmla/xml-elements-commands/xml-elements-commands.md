# <a name="commands-xmla"></a>Commandes (XMLA)
  Cette section de référence contient des éléments XML for Analysis (XMLA) qui peuvent être utilisés dans l'élément `Command` au cours d'un appel de méthode `Execute`.  
  
|Élément|Description|  
|-------------|-----------------|  
|[Alter, élément (XMLA)](alter-element-xmla.md)|Contient des éléments d’Analysis Services Scripting Language (ASSL) utilisés par le [Execute](../xml-elements-methods-execute.md) pour modifier les objets sur une instance de méthode [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../../../includes/ssasnoversion-md.md)].|  
|[Élément Backup](backup-element-xmla.md)|Sauvegarde une [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../../../includes/ssasnoversion-md.md)] base de données à un fichier de sauvegarde.|  
|[Élément de lot](batch-element-xmla.md)|Effectue XML d’un ou plusieurs pour les commandes Analysis (XMLA) comme un traitement par lots, dans l’ordre ou en parallèle, sur une instance de [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../../../includes/ssasnoversion-md.md)].|  
|[Élément BeginTransaction](begintransaction-element-xmla.md)|Démarre une transaction sur la session actuelle par une instance Analysis Services.|  
|[Élément Cancel](cancel-element-xmla.md)|Annule une commande en cours d'exécution dans une instance Analysis Services.|  
|[Élément ClearCache](clearcache-element-xmla.md)|Efface le cache mémoire de l'objet spécifié dans une instance Analysis Services.|  
|[Élément CommitTransaction](committransaction-element-xmla.md)|Valide une transaction dans la session actuelle avec une instance Analysis Services.|  
|[Créer l’élément](create-element-xmla.md)|Contient des éléments d’Analysis Services Scripting Language (ASSL) utilisés par le [Execute](../xml-elements-methods-execute.md) méthode pour créer des objets sur une instance Analysis Services.|  
|[Élément Delete](delete-element-xmla.md)|Supprime un objet dans une instance Analysis Services.|  
|[Élément DesignAggregations](designaggregations-element-xmla.md)|Crée des agrégations pour une conception d'agrégation dans une instance Analysis Services.|  
|[Élément DROP](drop-element-xmla.md)|Supprime des membres d'attribut d'une dimension.|  
|[Élément INSERT](insert-element-xmla.md)|Insère des membres d'attribut dans une dimension.|  
|[Élément Lock](lock-element-xmla.md)|Verrouille un objet spécifié dans une instance Analysis Services.|  
|[Élément MergePartitions](mergepartitions-element-xmla.md)|Fusionne les données d'une ou plusieurs partitions sources dans une partition cible, puis supprime les partitions sources.|  
|[Élément NotifyTableChange](notifytablechange-element-xmla.md)|Notifie une instance Analysis Services qu'une modification a été apportée à des tables dans une source de données spécifiée.|  
|[Élément Process](process-element-xmla.md)|Traite des objets sur une instance Analysis Services.|  
|[Élément Restore](restore-element-xmla.md)|Restaure une base de données Analysis Services à partir d'un fichier de sauvegarde.|  
|[Élément RollbackTransaction](rollbacktransaction-element-xmla.md)|Restaure une transaction dans la session en cours avec une instance Analysis Services.|  
|[Élément d’instruction](statement-element-xmla.md)|Contient une requête ou une instruction à envoyer à l’aide de la [Execute](../xml-elements-methods-execute.md) procédé à une instance d’Analysis Services.|  
|[Élément SUBSCRIBE](subscribe-element-xmla.md)|Procède à l'abonnement à une trace et retourne un ensemble de lignes contenant les événements de trace d'une instance Analysis Services.|  
|[Élément Synchronize](synchronize-element-xmla.md)|Synchronise une base de données Analysis Services avec une autre base de données existante.|  
|[Élément Unlock](unlock-element-xmla.md)|Déverrouille un verrou spécifié dans une instance Analysis Services.|  
|[Update (élément)](../xml-elements-commands/update-element-xmla.md)|Met à jour les membres d'attribut d'une dimension.|  
|[Élément UpdateCells](updatecells-element-xmla.md)|Met à jour des cellules dans un cube activé en écriture.|
