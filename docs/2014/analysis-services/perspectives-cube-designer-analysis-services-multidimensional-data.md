# <a name="perspectives-cube-designer-analysis-services---multidimensional-data"></a>Perspectives (Concepteur de cube) (Analysis Services - Données multidimensionnelles)
  Utilisez l'onglet **Perspectives** du Concepteur de cube pour créer et modifier les perspectives du cube sélectionné.  
  
 L'onglet **Perspectives** contient les volets suivants :  
  
## <a name="panes"></a>Volets  
 **Barre d'outils**  
 Permet d'utiliser les boutons de la barre d'outils pour effectuer les actions courantes relatives à l'onglet.  
  
 **Détails des perspectives**  
 Utilisez le volet des **détails des perspectives** pour créer et modifier les perspectives du cube sélectionné.
