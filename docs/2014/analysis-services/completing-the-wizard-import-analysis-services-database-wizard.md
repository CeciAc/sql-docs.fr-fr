# <a name="completing-the-wizard-import-analysis-services-database-wizard"></a>Fin de l'Assistant (Assistant Importation de base de données Analysis Services)
  La page **Fin de l'Assistant** permet d'afficher la progression de l'Assistant lors de l'importation des données et métadonnées de la base de données [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] sélectionnée dans le projet [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] .  
  
## <a name="options"></a>Options  
 **(Progression)**  
 Affiche la progression de l’Assistant à mesure que les objets de la base de données sont importés dans le projet [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] .
