# <a name="choose-time-calculations-business-intelligence-wizard"></a>Choisir les calculs de temps (Assistant Business Intelligence)
  Utilisez la page **Choisir les calculs de temps** pour définir la résolution des calculs de temps à créer pour une hiérarchie de temps existante.  
  
## <a name="options"></a>Options  
 **Utilisez la hiérarchie suivante pour analyser les calculs de temps**  
 Sélectionnez la hiérarchie de la dimension de temps à utiliser pour créer les calculs de temps.  
  
 **Calculs de temps disponibles**  
 Sélectionnez les calculs de temps à créer pour la hiérarchie sélectionnée.
