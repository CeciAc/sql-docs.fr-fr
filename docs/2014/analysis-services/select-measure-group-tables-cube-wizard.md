# <a name="select-measure-group-tables-cube-wizard"></a>Sélectionner les tables de groupes de mesures (Assistant Cube)
  Utilisez la page **Sélectionner les tables de groupes de mesures** pour identifier la vue de source de données et sélectionner les tables de groupes de mesures pour le cube.  
  
## <a name="options"></a>Options  
 **vue de source de données**  
 Permet de sélectionner la vue de source de données à utiliser pour le cube.  
  
 **Tables de groupes de mesures**  
 Permet de sélectionner les tables qui doivent être utilisées pour les groupes de mesures dans le cube.
