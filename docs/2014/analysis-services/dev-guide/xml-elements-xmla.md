# <a name="xml-elements-xmla"></a>Éléments XML (XMLA)
  Les rubriques suivantes décrivent le XML différents pour les catégories d’élément Analysis (XMLA) prises en charge par [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)].  
  
## <a name="in-this-section"></a>Dans cette section  
  
|Propriété|Description|  
|--------------|-----------------|  
|[En-têtes &#40;XMLA&#41;](../xmla/xml-elements-headers/xml-elements-headers.md)|Décrit les éléments envoyés dans l'en-tête SOAP d'une enveloppe SOAP soit par une application, soit par une instance [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)], pour gérer des fonctionnalités au niveau du protocole.|  
|[Méthodes &#40;XMLA&#41;](../xmla/xml-elements-methods.md)|Décrit les éléments les plus hauts envoyés par une application dans une enveloppe SOAP à une instance [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)] pour récupérer des données ou des métadonnées, ou effectuer des actions sur cette instance.|  
|[Commandes &#40;XMLA&#41;](../xmla/xml-elements-commands/xml-elements-commands.md)|Décrit les éléments envoyés dans une méthode XMLA pour effectuer une action spécifique.|  
|[Objets &#40;XMLA&#41;](../xmla/xml-elements-objects.md)|Décrit les éléments les plus hauts reçus par une application dans une enveloppe SOAP émanant d'une instance [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)] en réponse à une méthode XMLA.|  
|[Propriétés &#40;XMLA&#41;](../xmla/xml-elements-properties/xml-elements-properties.md)|Décrit les éléments enfants pour les en-têtes, les méthodes, les objets ou les commandes XMLA.|
