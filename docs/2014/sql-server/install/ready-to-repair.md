# <a name="ready-to-repair"></a>Prêt à réparer
  La page Prêt à réparer fournit un résumé du processus de réparation par le programme d’installation de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
## <a name="options"></a>Options  
 La page affiche les options suivantes :  
  
-   Pour continuer l'installation, cliquez sur **Réparer**.  
  
-   Pour modifier vos paramètres d'installation, cliquez sur **Précédent**.  
  
-   Pour quitter le programme d'installation, cliquez sur **Annuler**.
