# <a name="use-the-distributed-replay-utility-to-prepare-for-upgrades"></a>Utiliser Distributed Replay Utility pour préparer des mises à niveau
  Le [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Distributed Replay Utility peut utiliser plusieurs ordinateurs pour relire les données de trace, simulant une charge de travail stratégiques. En effectuant une nouvelle lecture sur un serveur de test avant et après une mise à niveau de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)], vous pouvez mesurer les différences de performance et rechercher toute incompatibilité entre votre application et la mise à niveau. Pour plus d'informations, consultez [SQL Server Distributed Replay](../../tools/distributed-replay/sql-server-distributed-replay.md).
