# <a name="ready-to-prepare-image"></a>Prêt à préparer l'image
  La page **Prêt à préparer l'image** fournit un résumé des fonctionnalités devant être préparées par le programme d'installation de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .  
  
## <a name="options"></a>Options  
 La page affiche les options suivantes :  
  
-   Pour continuer l'installation, cliquez sur **Installer**.  
  
-   Pour modifier vos paramètres d'installation, cliquez sur **Précédent**.  
  
-   Pour quitter le programme d'installation, cliquez sur **Annuler**.
