# <a name="complete---upgrade"></a>Complète - Mise à niveau
  Utilisez la page Complète (Mise à niveau) pour vérifier que la mise à niveau est terminée.  
  
## <a name="options"></a>Options  
 Cette boîte de dialogue comprend un lien vers le fichier journal d'installation pour cette opération. Pour continuer, cliquez sur **Fermer**.
