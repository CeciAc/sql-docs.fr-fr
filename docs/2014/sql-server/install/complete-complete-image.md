# <a name="complete---complete-image"></a>Finaliser - Finaliser l'image
  Utilisez la page **Complète** pour vérifier que la configuration de l'instance préparée est complète.  
  
## <a name="options"></a>Options  
 Cette boîte de dialogue comprend un lien vers le fichier journal d'installation pour cette opération. Pour terminer l'installation, cliquez sur **Fermer**.
