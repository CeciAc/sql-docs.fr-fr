# <a name="prepare-image-progress"></a>Progression de la préparation de l'image

À l'issue de cette étape, cette page répertorie l'état de chaque fonctionnalité [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] préparée durant ce processus.
