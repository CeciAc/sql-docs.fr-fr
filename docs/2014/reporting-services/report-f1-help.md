# <a name="report-f1-help"></a>Aide F1 sur les rapports
  Les rubriques de cette section fournissent des informations sur les options que vous pouvez définir sur le rapport dans la boîte de dialogue **Propriétés du rapport** . Pour ouvrir la boîte de dialogue **Propriétés du rapport** , cliquez avec le bouton droit sur l'arrière-plan du Concepteur de rapports, puis sur **Propriétés du rapport**.  
  
> [!NOTE]  
>  Il est facile de confondre le corps du rapport avec le rapport. Le corps du rapport est l'élément du rapport qui contient le contenu du rapport. Le rapport inclut le corps du rapport et, éventuellement, une en-tête et un pied de page. Si vous cliquez avec le bouton droit sur le corps du rapport, vous ne pouvez pas accéder ou définir des propriétés de rapport.  
  
 Pour accéder à ces rubriques depuis la boîte de dialogue **Propriétés du rapport** , cliquez **Aide**.
