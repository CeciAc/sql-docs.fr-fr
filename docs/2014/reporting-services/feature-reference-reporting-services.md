# <a name="feature-reference-reporting-services"></a>Références relatives aux fonctionnalités (Reporting Services)
  [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssRSnoversion](../includes/ssrsnoversion-md.md)] contient plusieurs outils et applications qui permettent de créer, gérer et afficher des rapports. Cette section répertorie les rubriques spécifiques qui décrivent les boîtes de dialogue, les pages Web et les Assistants de ces outils et applications.  
  
 Pour accéder à une rubrique de l’interface utilisateur pendant l’exécution de l’outil ou de l’application, appuyez sur F1 ou cliquez sur **Aide** lorsque la boîte de dialogue, la page web ou l’Assistant est ouvert. Pour plus d’informations sur le démarrage de la [!INCLUDE[ssRSnoversion](../includes/ssrsnoversion-md.md)] tools, consultez [didacticiel : Comment localiser et démarrer Reporting Services outils &#40;SSRS&#41;](tools/tutorial-how-to-locate-and-start-reporting-services-tools-ssrs.md).
