# <a name="complete-the-report-wizard"></a>Terminer l'Assistant Rapport
  Utilisez cette page pour nommer le rapport et passer en revue les options qui doivent être appliquées.  
  
## <a name="options"></a>Options  
 **Nom du rapport**  
 Entrez un nom pour le rapport. Ce nom apparaîtra également comme titre de votre rapport. Vous pouvez modifier le titre en mode Création.  
  
 **Résumé du rapport**  
 Consultez les options sélectionnées au cours des précédentes étapes de l'Assistant Rapport. Pour modifier une option, cliquez sur **Précédent** jusqu'à ce que la page contenant l'option à modifier s'affiche.  
  
 **Aperçu du rapport**  
 Choisissez cette option pour afficher un aperçu du rapport que vous avez créé. Sinon, il s'ouvrira en mode Création.
