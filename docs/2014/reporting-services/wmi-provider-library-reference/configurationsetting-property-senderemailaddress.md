# <a name="senderemailaddress-property-wmi-msreportserverconfigurationsetting"></a>Propriété SenderEmailAddress (WMI MSReportServer_ConfigurationSetting)
  Obtient l'adresse utilisée pour l'envoi de messages électroniques à partir du serveur de rapports. En lecture seule.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
Public Dim SenderEmailAddress As String  
```  
  
```csharp  
public string SenderEmailAddress;  
```  
  
## <a name="property-values"></a>Valeurs de la propriété  
 Objet `String` en lecture seule qui représente l'adresse de messagerie utilisé par le serveur de rapports.
