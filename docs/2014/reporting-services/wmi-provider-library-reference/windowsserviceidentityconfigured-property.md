# <a name="windowsserviceidentityconfigured-property-wmi-msreportserverconfigurationsetting"></a>Propriété WindowsServiceIdentityConfigured (WMI MSReportServer_ConfigurationSetting)
  Retourne l'identité sous laquelle le service Windows Report Server a été configuré la dernière fois pour son exécution. En lecture seule.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
Public Dim WindowsServiceIdentityConfigured As String  
```  
  
```csharp  
public string WindowsServiceIdentityConfigured;  
```  
  
## <a name="property-values"></a>Valeurs de la propriété  
 Valeur `String` contenant l'identité sous laquelle le service Windows Report Server a été configuré la dernière fois pour son exécution.
