# <a name="instancename-property-wmi-msreportserverinstance"></a>Propriété InstanceName (WMI MSReportServer_Instance)
  Retourne le nom d'une instance du serveur de rapports sur un ordinateur spécifique. En lecture seule.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
Public Dim InstanceName As String  
```  
  
```csharp  
public string InstanceName;  
```  
  
## <a name="property-value"></a>Valeur de propriété  
 Objet `String` qui représente le nom de l'instance de serveur de rapports.
