# <a name="isinitialized-property-wmi-msreportserverconfigurationsetting"></a>Propriété IsInitialized (WMI MSReportServer_ConfigurationSetting)
  Indique si l'instance de serveur de rapports est initialisée. En lecture seule.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
Public Dim IsInitialized As Boolean  
```  
  
```csharp  
public Boolean DatabaseLogonName;  
```  
  
## <a name="property-values"></a>Valeurs de la propriété  
 Objet `Boolean` indiquant si l'instance de serveur de rapports est initialisée.
