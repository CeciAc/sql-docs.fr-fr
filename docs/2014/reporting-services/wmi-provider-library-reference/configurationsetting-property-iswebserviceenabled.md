# <a name="iswebserviceenabled-property-wmi-msreportserverconfigurationsetting"></a>Propriété IsWebServiceEnabled (WMI MSReportServer_ConfigurationSetting)
  Indique si le service Web Report Server est activé. En lecture seule.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
Public Dim IsWebServiceEnabled As Boolean  
```  
  
```csharp  
public boolean IsWebServiceEnabled;  
```  
  
## <a name="property-values"></a>Valeurs de la propriété  
 Valeur `Boolean` en lecture seule. La valeur `true` indique que le service Web Report Server est activé.
