# <a name="version-property--wmi-msreportserverconfigurationsetting"></a>Propriété Version (WMI MSReportServer_ConfigurationSetting)
  Retourne la version du serveur de rapports au format Major.Minor.Build.Revision. En lecture seule.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
Public Dim Version As String  
```  
  
```csharp  
public string Version;  
```  
  
## <a name="property-value"></a>Valeur de propriété  
 Un `string` qui contient la version du serveur de rapports.
