# <a name="machineaccountidentity-property-wmi"></a>Propriété MachineAccountIdentity (WMI)
  Obtient l'identité du compte d'ordinateur de l'ordinateur sur lequel le serveur de rapports est installé.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
Public Dim MachineAccountIdentity As String  
```  
  
```csharp  
public string MachineAccountIdentity;  
```  
  
## <a name="property-values"></a>Valeurs de la propriété  
 A `String` valeur qui contient l’identité de compte d’ordinateur de l’ordinateur installé sur le serveur de rapports.
