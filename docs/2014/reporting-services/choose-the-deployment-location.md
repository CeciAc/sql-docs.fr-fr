# <a name="choose-the-deployment-location"></a>Choisir l'emplacement du déploiement
  Utilisez cette page de l'Assistant Rapport pour définir le serveur et le dossier de déploiement du projet Report Server.  
  
## <a name="options"></a>Options  
 **Serveur de rapports**  
 Tapez l’URL du répertoire virtuel du serveur de rapports, par exemple, http://localhost/reportserver. Il s'agit du répertoire virtuel du serveur de rapports et non du Gestionnaire de rapports. Par défaut, lors de l'installation du serveur de rapports, un répertoire virtuel nommé reportserver est créé.  
  
 **Dossier de déploiement**  
 Tapez le nom du dossier de destination pour la publication des rapports contenus dans le projet Report Server. Par défaut, le nom du dossier correspond au nom du projet. Si le dossier n'existe pas sur le serveur de rapports, le Générateur de rapports le crée lors de la publication des rapports.
