Parties de rapports sont self\-contenait des éléments de rapport qui sont stockés sur le serveur de rapports et peuvent être inclus dans d’autres rapports. Utilisez le Générateur de rapports pour parcourir et sélectionner des parties de rapports dans la bibliothèque de parties de rapports afin de les ajouter à vos rapports. Utilisez le Concepteur de rapports ou le Générateur de rapports pour enregistrer des parties de rapports à utiliser dans la bibliothèque de parties de rapports. Pour plus d’informations, consultez **parties de rapports \(Générateur de rapports et SSRS\)**  et **des parties de rapports dans le Concepteur de rapports \(SSRS\)**  sur le site Web Microsoft.com.
