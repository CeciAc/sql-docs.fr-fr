# <a name="summary-writing-transact-sql"></a>Résumé : Écriture d'instructions Transact-SQL
  L'Éditeur de requête permet de franchir une étape importante vers l'utilisation d'un environnement d'auteur sophistiqué dans le cadre du développement [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] . Vous pouvez facilement afficher l'Éditeur de requête en mode plein écran pour visualiser votre code ou changer rapidement votre connexion pour l'établir à un autre serveur. La boîte de dialogue **Options** accessible à partir du menu **Outils** est l'endroit le plus approprié pour rechercher des fonctions supplémentaires.
