# <a name="trace-file-properties-general-tab"></a>Propriétés du fichier de trace (onglet Général)
  Utilisez l'onglet **Général** de la boîte de dialogue **Propriétés du fichier de trace** pour consulter les propriétés d'un fichier de trace.  
  
 Pour afficher cette fenêtre, ouvrez un fichier de trace. Ensuite, dans le menu **Fichier** , cliquez sur **Propriétés**.  
  
## <a name="options"></a>Options  
 **Nom de fichier**  
 Le chemin d'accès et le nom du fichier de trace ouvert.  
  
 **Nom du fournisseur de trace**  
 Affiche le nom de l'instance [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] qui a été tracée.  
  
 **Type de fournisseur de trace**  
 Affiche le type de serveur qui a fourni la trace.  
  
 **version**  
 Affiche la version du serveur qui a fourni la trace.  
  
 **Taille du fichier (Ko)**  
 La taille du fichier de trace en kilo-octets (Ko).  
  
 **Créé le**  
 La date et l'heure de création du fichier de trace.  
  
 **Modifié le**  
 La date et l'heure de modification du fichier de trace.
