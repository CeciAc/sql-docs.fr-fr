# <a name="architecture-of-clr-integration"></a>Architecture d'intégration du CLR
  L'intégration [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] avec le Common Language Runtime (CLR) .NET Framework permet aux programmeurs de base de données d'utiliser des langages tels que Visual C#, Visual Basic .NET et Visual C++. Les fonctions, procédures stockées, déclencheurs, types de données et agrégats sont parmi les types de logique métier que les programmeurs peuvent écrire avec ces langages.  
  
 Cette section décrit l'architecture d'intégration du CLR à l'intérieur de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] et la façon dont cette architecture fournit un environnement sûr, évolutif, sécurisé et efficace pour exécuter du code utilisateur.

 Discute des objectifs de conception architecturale, des mécanismes et des avantages offerts par l'intégration du CLR.  
  
 [Performances de l’intégration du CLR](../../relational-databases/clr-integration/clr-integration-architecture-performance.md)  
 Traite des considérations relatives aux performance de l'architecture d'intégration du CLR.
