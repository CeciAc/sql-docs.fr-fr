# <a name="system-requirements"></a>Configuration système requise
  [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Management Objects (SMO) utilise le [!INCLUDE[msCoName](../../includes/msconame-md.md)] `System.Data.SqlClient` pilote d’objet de connexion et la communication avec des instances de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
 Clients SMO nécessitent [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client, qui est inclus avec [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] et [!INCLUDE[dnprdnshort](../../includes/dnprdnshort-md.md)] 2.0.  
  
 Pour développer des applications à l’aide de SMO, vous devez avoir [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] 2008 est installé.
