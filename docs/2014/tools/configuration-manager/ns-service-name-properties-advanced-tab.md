# <a name="nsltservice-namegt-properties-advanced-tab"></a>Propriétés de NS$&lt;nom du service&gt; (onglet Avancé)
  [!INCLUDE[ssNS](../../includes/ssns-md.md)] est implémenté en tant que service nommé au format **NS$**_< nom configuré_utilisateur >_. Si des propriétés personnalisées sont définies, elles apparaissent dans cet onglet, avec les valeurs correspondantes. Il est possible qu'aucune propriété ne soit répertoriée dans cet onglet.
