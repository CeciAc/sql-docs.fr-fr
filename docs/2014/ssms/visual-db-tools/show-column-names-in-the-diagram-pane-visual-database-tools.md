# <a name="show-column-names-in-the-diagram-pane-visual-database-tools"></a>Afficher les noms des colonnes dans le volet Schéma (Visual Database Tools)
  Vous pouvez choisir de masquer ou d’afficher les noms des colonnes des tables dans le **volet Schéma** du Concepteur de requêtes et de vues.  
  
### <a name="show-column-names"></a>Affichage des noms des colonnes  
  
1.  Cliquez avec le bouton droit sur la table dans le **volet Schéma** du Concepteur de requêtes et de vues.  
  
2.  Dans le menu contextuel, sélectionnez **Noms des colonnes** .  
  
### <a name="hide-column-names"></a>Masquage des noms des colonnes  
  
1.  Cliquez avec le bouton droit sur la table dans le **volet Schéma** du Concepteur de requêtes et de vues.  
  
2.  Dans le menu contextuel, sélectionnez **Nom uniquement** .
