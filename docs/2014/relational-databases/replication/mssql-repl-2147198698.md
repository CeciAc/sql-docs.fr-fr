# <a name="mssqlrepl-2147198698"></a>MSSQL_REPL-2147198698
    
## <a name="message-details"></a>Détails du message  
  
|||  
|-|-|  
|Nom du produit|SQL Server|  
|ID d'événement|-2147198698|  
|Source de l'événement|MSSQLServer|  
|Nom symbolique||  
|Texte du message|L'instantané de cette publication est devenu obsolète. L'agent d'instantané doit être à nouveau exécuté avant que l'abonnement puisse être synchronisé.|  
  
## <a name="explanation"></a>Explication  
 L'instantané de cette publication est devenu obsolète.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 L'Agent d'instantané doit être réexécuté pour permettre la synchronisation de l'abonnement.
