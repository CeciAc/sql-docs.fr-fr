# <a name="snapshot-options"></a>Options d'instantané
  Il existe de nombreuses options disponibles lors de l'initialisation d'un abonnement avec un instantané :  
  
-   Spécifier un emplacement de dossier d'instantanés de remplacement à la place ou en plus de l'emplacement de dossier d'instantanés par défaut. Pour plus d’informations, consultez [Alternate Snapshot Folder Locations](alternate-snapshot-folder-locations.md).  
  
-   Compresser les instantanés pour un stockage sur support amovible ou pour un transfert sur un réseau lent. Pour plus d’informations, voir [Compressed Snapshots](compressed-snapshots.md).  
  
-   Exécuter des scripts [!INCLUDE[tsql](../../includes/tsql-md.md)] avant ou après l'application de l'instantané. Pour plus d’informations, consultez [Exécuter des scripts avant et après l’application de l’instantané](execute-scripts-before-and-after-the-snapshot-is-applied.md).  
  
-   Transférer des fichiers d'instantanés via le protocole FTP. Pour plus d’informations, consultez [Transférer des instantanés via FTP](transfer-snapshots-through-ftp.md).
