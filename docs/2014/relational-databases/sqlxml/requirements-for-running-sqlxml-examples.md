# <a name="requirements-for-running-sqlxml-examples"></a>Configuration requise pour l'exécution des exemples SQLXML
  Le [programmation SQLXML 4.0](sqlxml-4-0-programming-concepts.md) section apporte de nombreux exemples. Cette rubrique répertorie les éléments requis pour la création d'exemples fonctionnels à partir de ces exemples.  
  
 Pour créer des exemples fonctionnels à l'aide des exemples de SQLXML 4.0, vous devez avoir installé les composants suivants.  
  
-   Exemple de base de données AdventureWorks. Pour plus d’informations, consultez [AdventureWorks Sample Databases](http://msftdbprodsamples.codeplex.com/).  
  
-   Microsoft [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client. Pour plus d’informations, consultez [Installation de SQL Server Native Client](../native-client/applications/installing-sql-server-native-client.md).  
  
-   MDAC 2.6 ou version ultérieure  
  
 Dans de nombreux exemples, les modèles sont utilisés pour spécifier des requêtes XPath par rapport au schéma de mappage XSD. Pour plus d’informations, consultez [à l’aide des schémas XSD annotés dans les requêtes &#40;SQLXML 4.0&#41;](annotated-xsd-schemas/using-annotated-xsd-schemas-in-queries-sqlxml-4-0.md).
