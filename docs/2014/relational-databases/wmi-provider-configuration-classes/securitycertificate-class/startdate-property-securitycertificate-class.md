# <a name="startdate-property-securitycertificate-class"></a>Propriété StartDate (classe SecurityCertificate)
  Obtient la date à laquelle le certificat de sécurité entre en vigueur.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
object  
.StartDate [= value]  
```  
  
## <a name="parts"></a>Éléments  
 *object*  
 Un [classe SecurityCertificate] securitycertificate-class.md) objet qui représente un certificat de sécurité.  
  
## <a name="property-valuereturn-value"></a>Valeur de propriété/valeur de retour  
 Valeur uint32 qui spécifie la date à laquelle le certificat de sécurité entre en vigueur.
