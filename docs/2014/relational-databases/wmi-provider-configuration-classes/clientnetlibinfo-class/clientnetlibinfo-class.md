# <a name="clientnetlibinfo-class"></a>Classe ClientNetLibInfo
  La [classe ClientNetLibInfo](clientnetlibinfo-class.md) représente des informations sur la bibliothèque réseau cliente.  
  
 Le fichier de bibliothèque réseau cliente permet à l'instance du client de communiquer sur un réseau avec une instance de [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] à l'aide d'un protocole réseau spécifique.
