# <a name="propertynumval-property-clientnetworkprotocolproperty-class"></a>Propriété PropertyNumVal (classe ClientNetworkProtocolProperty)
  Obtient la valeur numérique de la propriété actuelle référencée par la [propertyidx, propriété (classe ClientNetworkProtocolProperty)](clientnetworkprotocolproperty-class.md) valeur.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
object  
.PropertyNumVal [= value]  
```  
  
## <a name="parts"></a>Éléments  
 *object*  
 Objet de [classe ClientNetworkProtocolProperty](clientnetworkprotocolproperty-class.md) qui représente un attribut du protocole réseau utilisé par le client [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] .  
  
## <a name="property-valuereturn-value"></a>Valeur de propriété/valeur de retour  
 Valeur u`int32` qui spécifie la valeur numérique de la propriété actuelle.
