# <a name="sinstance-class"></a>Classe SInstance
  La [classe SInstance](sinstance-class.md) représente une instance de [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
 La classe [SInstance Class](sinstance-class.md) peut être utilisée pour gérer les protocoles et les bibliothèques réseau associés à l'instance de [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]. Un certificat de sécurité peut également être spécifié pour l'instance de [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].
