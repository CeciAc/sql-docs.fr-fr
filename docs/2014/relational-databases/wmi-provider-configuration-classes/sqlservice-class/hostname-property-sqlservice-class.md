# <a name="hostname-property-sqlservice-class"></a>Propriété HostName (classe SqlService)
  Obtient le nom d'hôte du service.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
object  
.HostName [= value]  
```  
  
## <a name="parts"></a>Éléments  
 *object*  
 Objet de [classe SqlService](sqlservice-class.md) qui représente le service.  
  
## <a name="property-valuereturn-value"></a>Valeur de propriété/valeur de retour  
 Valeur de chaîne qui spécifie le nom d'hôte du service.
