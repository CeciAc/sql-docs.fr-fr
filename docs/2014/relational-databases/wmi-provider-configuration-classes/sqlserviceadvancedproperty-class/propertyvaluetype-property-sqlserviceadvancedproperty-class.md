# <a name="propertyvaluetype-property-sqlserviceadvancedproperty-class"></a>Propriété PropertyValueType (classe SqlServiceAdvancedProperty)
  Obtient le type de données d'une propriété.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
object  
.PropertyValueType [= value]  
```  
  
## <a name="parts"></a>Éléments  
 *object*  
 Objet de [classe SqlServiceAdvancedProperty](sqlserviceadvancedproperty-class.md) qui représente une propriété avancée.  
  
## <a name="property-valuereturn-value"></a>Valeur de propriété/valeur de retour  
 Valeur `uint32` qui représente le type de données de la propriété.
