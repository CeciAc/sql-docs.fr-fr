# <a name="summary-managing-data-in-a-hierarchical-table"></a>Résumé : Gestion de données dans une table hiérarchique
  Le type de données `hierarchyid` vous permet de créer facilement des tables qui reflètent des données hiérarchiques, puis de créer des index à profondeur prioritaire et à largeur prioritaire pour accéder de manière efficace à la table. Utilisez les méthodes hiérarchiques pour interroger et manipuler les données.
