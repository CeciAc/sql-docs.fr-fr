# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
  L’interface IBCPSession2 est une extension IBCPSession qui fournit une fonction membre qui est une alternative à l’appel IBCPSession::BCPColFmt pour chaque colonne.  IBCPSession2 hérite IBCPSession et ajoute une nouvelle méthode : [IBCPSession2::BCPSetBulkMode](ibcpsession2-bcpsetbulkmode.md).
