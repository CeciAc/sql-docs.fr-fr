# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
  Ibcpsession2, de l’interface est une extension IBCPSession qui fournit une fonction membre qui est une alternative à l’appel de IBCPSession::BCPColFmt pour chaque colonne.  IBCPSession2 hérite d’IBCPSession et ajoute une nouvelle méthode : [IBCPSession2::BCPSetBulkMode](ibcpsession2-bcpsetbulkmode.md).
