# <a name="localdberrorcannotmodifyinstanceregistry"></a>LOCALDB_ERROR_CANNOT_MODIFY_INSTANCE_REGISTRY
    
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|SQL Server|  
|ID d'événement|265|  
|Source de l'événement|Runtime de base de données locale SQL Server 12.0|  
|Composant|API d'exécution de la base de données locale|  
|Texte du message|Une erreur inattendue s'est produite lors de la tentative de modifier la configuration du Registre pour l'instance de base de données locale. Pour plus d'informations sur l'erreur, consultez le journal des applications Windows.|  
  
## <a name="explanation"></a>Explication  
 Impossible de modifier un Registre d'instance.
