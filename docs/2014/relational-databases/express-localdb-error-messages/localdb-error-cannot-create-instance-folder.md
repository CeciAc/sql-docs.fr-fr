# <a name="localdberrorcannotcreateinstancefolder"></a>LOCALDB_ERROR_CANNOT_CREATE_INSTANCE_FOLDER
    
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|SQL Server|  
|ID d'événement|256|  
|Source de l'événement|Runtime de base de données locale SQL Server 12.0|  
|Composant|API d'exécution de la base de données locale|  
|Texte du message|Impossible de créer le dossier pour l’instance de base de données locale dans : %%LOCALAPPDATA%%\Microsoft\Microsoft SQL Server Local DB\Instances\\< nom de l’instance\>.|  
  
## <a name="explanation"></a>Explication  
 Impossible de créer un dossier sous %userprofile%.
