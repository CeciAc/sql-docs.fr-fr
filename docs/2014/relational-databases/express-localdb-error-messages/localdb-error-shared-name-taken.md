# <a name="localdberrorsharednametaken"></a>LOCALDB_ERROR_SHARED_NAME_TAKEN
    
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|SQL Server|  
|ID d'événement|281|  
|Source de l'événement|Runtime de base de données locale SQL Server 12.0|  
|Composant|API d'exécution de la base de données locale|  
|Texte du message|Le nom partagé spécifié pour une instance de base de données locale est déjà utilisé.|  
  
## <a name="explanation"></a>Explication  
 Le nom partagé spécifié pour une instance de base de données locale est déjà utilisé.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Essayez un autre nom pour l'instance d'exécution de base de données locale ou supprimez l'instance existante.
