# <a name="localdberroradminrightsrequired"></a>LOCALDB_ERROR_ADMIN_RIGHTS_REQUIRED
    
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|SQL Server|  
|ID d'événement|286|  
|Source de l'événement|Runtime de base de données locale SQL Server 12.0|  
|Composant|API d'exécution de la base de données locale|  
|Texte du message|Des privilèges d'administrateur sont requis pour effectuer cette opération.|  
  
## <a name="explanation"></a>Explication  
 L'opération demandée peut être exécutée uniquement par l'utilisateur doté d'autorisations d'administrateur.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Contactez l'administrateur de l'ordinateur pour obtenir de l'aide.
