# <a name="localdberrorinstanceconfigurationcorrupt"></a>LOCALDB_ERROR_INSTANCE_CONFIGURATION_CORRUPT
    
## <a name="details"></a>Détails  
  
|||  
|-|-|  
|Nom du produit|SQL Server|  
|ID d'événement|267|  
|Source de l'événement|Runtime de base de données locale SQL Server 12.0|  
|Composant|API d'exécution de la base de données locale|  
|Texte du message|L'instance de base de données locale est endommagée. Pour plus d'informations sur l'erreur, consultez le journal des applications Windows.|  
  
## <a name="explanation"></a>Explication  
 Une configuration d'instance est endommagée.  
  
## <a name="user-action"></a>Action de l'utilisateur  
 Corrigez manuellement ou supprimez explicitement l'instance endommagée.
