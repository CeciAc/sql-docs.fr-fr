# <a name="errors-and-events-reference-database-engine"></a>Références relatives aux erreurs et aux événements (moteur de base de données)
  Le [Events and Errors Message Center](http://go.microsoft.com/fwlink/?LinkId=47660) sur TechNet constitue la principale source d’informations sur les messages d’erreur pour [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. Ils améliorent l'expérience du client lors de la recherche d'informations sur les messages d'erreur de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] et d'autres produits Microsoft en lui permettant d'effectuer des recherches en texte intégral et de filtrer les résultats en fonction d'un produit, d'une version, d'une source ou d'une identification numérique ou symbolique. En outre, la nouvelle interface assure la prise en charge de plusieurs langues. Le Centre sera continuellement mis à jour avec les informations de dépannage concernant les erreurs les plus fréquentes.  
  
 Des améliorations seront prochainement apportées à ce centre pour proposer toutes les informations de dépannage sous forme téléchargeable afin de pouvoir y accéder à partir d'un ordinateur non connecté à Internet.
