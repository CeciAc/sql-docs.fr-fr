# <a name="internet-information-services-error-codes-for-remote-data-service"></a>Codes d’erreur des Services IIS pour le Service de données distant
Le tableau suivant répertorie les codes d’erreur de Microsoft® Internet Information Services (IIS) liés à l’utilisation du Service de données distant. La conversion décimale positive des deux octets faible, la conversion décimale négative du code d’erreur complet et les valeurs hexadécimales sont affichés.

|Erreurs de Internet Information Services|Number|Description|
|------------------------------------------|------------|-----------------|
|**IDS_IIS_AccessDenied**|8208 -2146820080 0x800A2010|Erreur du serveur Internet : Accès refusé.|
|**IDS_IIS_ObjectNotFound**|8209 -2146820079 0x800A2011|Erreur du serveur Internet : Objet de module introuvable.|
|**IDS_IIS_RequestForbidden**|8210 -2146820078 0x800A2012|Erreur du serveur Internet : Il est interdite de demande.|
|**IDS_IIS_UnexpectedError**|8447 -2146819841 0x800A20FF|Erreur du serveur Internet.|
