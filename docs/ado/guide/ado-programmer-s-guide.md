# <a name="ado-programmer39s-guide-for-using-ado-objects"></a>Programmeur ADO&#39;s Guide pour l’utilisation des objets ADO
Cette documentation fournit une vue d’ensemble de l’utilisation des objets ADO pour travailler avec des données à partir de diverses sources de données. Tout d’abord, l’introduction résume les nouvelles fonctionnalités et la configuration requise pour l’installation, à l’aide et redistribuer le composant ADO. Les sections suivantes décrivent l’utilisation des différents types d’objets ADO. Enfin, les annexes décrivent les différentes données fournis par Microsoft et les fournisseurs de services, les messages d’erreur et les conseils sur l’utilisation d’ADO dans plusieurs langages de programmation, en plus des exemples et un glossaire.
