# <a name="understanding-recordset-structure"></a>Présentation de la structure d’un recordset
Chaque **Recordset** a un **champs** collection composée d’un ou plusieurs **champ** objets. Un **champ** objet représente généralement une colonne de table. Les rubriques suivantes expliquent comment naviguer dans le **champs** collection et obtenez des informations sur chaque champ. Alors elles présentent le type d’informations est disponible pour vous par le biais de la **champ** objet et comment l’utiliser.  
  
-   [Fields, collection](../../../ado/guide/data/the-fields-collection.md)  
  
-   [Field, objet](../../../ado/guide/data/the-field-object.md)  
  
-   [Utilisation des recordsets](../../../ado/guide/data/working-with-recordsets.md)
