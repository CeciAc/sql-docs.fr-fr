# <a name="source-property-ado-md"></a>Source, propriété (ADO MD)
Indique la source pour les données dans le [cellset](../../../ado/reference/ado-md-api/cellset-object-ado-md.md).  
  
## <a name="settings-and-return-values"></a>Paramètres et valeurs de retour  
 Définit ou retourne un **Variant**, et est en lecture/écriture pour fermé [Cellset](../../../ado/reference/ado-md-api/cellset-object-ado-md.md) objets et en lecture seule pour open **Cellset** objets. Le **Variant** doit contenir un valide **chaîne**, par exemple, une requête MDX.
