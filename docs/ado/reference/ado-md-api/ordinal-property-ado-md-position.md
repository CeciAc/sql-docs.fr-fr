# <a name="ordinal-property-ado-md-position"></a>Ordinal, propriété (objet Position d’ADO MD)
Identifie de façon unique un [position](../../../ado/reference/ado-md-api/position-object-ado-md.md) long d’un axe.  
  
## <a name="return-values"></a>Valeurs de retour  
 Retourne un **Long** entier et est en lecture seule.  
  
## <a name="remarks"></a>Notes  
 Le **Ordinal** propriété d’un [Position](../../../ado/reference/ado-md-api/position-object-ado-md.md) objet correspond à l’index de la **Position** au sein de la [Positions](../../../ado/reference/ado-md-api/positions-collection-ado-md.md) collection.  
  
 Une cellule peut être récupérée rapidement à l’aide de la **Ordinal** de la **Position** le long de chaque axe avec la [élément](../../../ado/reference/ado-md-api/item-property-ado-md-cellset.md) propriété de la [Cellset](../../../ado/reference/ado-md-api/cellset-object-ado-md.md) objet .
