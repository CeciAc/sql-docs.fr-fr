# <a name="command-property-adox"></a>Command, propriété (ADOX)
Spécifie une ADO [commande](../../../ado/reference/ado-api/command-object-ado.md) objet qui peut être utilisé pour créer ou exécuter la procédure.  
  
## <a name="settings-and-return-values"></a>Paramètres et valeurs de retour  
 Définit ou renvoie un ADO valide **commande** objet.  
  
## <a name="remarks"></a>Notes  
 Une erreur se produit lors de l’obtention et la définition de cette propriété si le fournisseur ne prend pas en charge les commandes persistantes.
