# <a name="syspdwdiageventproperties-transact-sql"></a>Sys.pdw_diag_event_properties (Transact-SQL)
[!INCLUDE[tsql-appliesto-xxxxxx-xxxx-xxxx-pdw-md](../../includes/tsql-appliesto-xxxxxx-xxxx-xxxx-pdw-md.md)]

  Contient des informations sur les propriétés sont associées à des événements de diagnostic.  
  
|Nom de la colonne|Type de données|Description|Plage|  
|-----------------|---------------|-----------------|-----------|  
|**event_name**|**nvarchar(255)**|Nom de l’événement de diagnostic spécifiques.||  
|**property_name**|**nvarchar(255)**|Nom d’une propriété de l’événement.||
