# <a name="client-side-and-server-side-formatting-sqlxml-40"></a>Mise en forme côté client et côté serveur (SQLXML 4.0)
[!INCLUDE[appliesto-ss-asdb-xxxx-xxx-md](../../../includes/appliesto-ss-asdb-xxxx-xxx-md.md)]
  SQLXML propose deux manières de mettre en forme un document XML :  
  
-   Mise en forme côté serveur  
  
-   Mise en forme côté client  
  
 Cette section inclut des informations sur la mise en forme de documents XML sur le côté client ou côté serveur des ensembles de lignes qui sont générés par les requêtes exécutées sur une base de données [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].
