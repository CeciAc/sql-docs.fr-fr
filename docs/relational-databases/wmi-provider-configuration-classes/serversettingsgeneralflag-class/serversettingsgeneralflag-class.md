# <a name="serversettingsgeneralflag-class"></a>Classe ServerSettingsGeneralFlag
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  La [classe ServerSettingsGeneralFlag](../../../relational-databases/wmi-provider-configuration-classes/serversettingsgeneralflag-class/serversettingsgeneralflag-class.md) représente les paramètres de l'indicateur général sur une instance du client [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
 Cette classe peut servir à effectuer les tâches suivantes :  
  
-   afficher le nom de l'indicateur ;  
  
-   afficher et modifier la valeur de l'indicateur.
