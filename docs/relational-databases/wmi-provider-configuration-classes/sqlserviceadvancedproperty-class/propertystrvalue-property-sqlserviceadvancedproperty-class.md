# <a name="propertystrvalue-property-sqlserviceadvancedproperty-class"></a>Propriété PropertyStrValue (classe SqlServiceAdvancedProperty)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  Obtient la valeur de chaîne de la propriété avancée.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
object.PropertyStrValue [= value]  
```  
  
## <a name="parts"></a>Éléments  
 *object*  
 Un [classe SqlServiceAdvancedProperty](../../../relational-databases/wmi-provider-configuration-classes/sqlserviceadvancedproperty-class/sqlserviceadvancedproperty-class.md) objet qui représente une propriété avancée.  
  
## <a name="property-valuereturn-value"></a>Valeur de propriété/valeur de retour  
 Valeur de chaîne qui spécifie la valeur de la propriété avancée.
