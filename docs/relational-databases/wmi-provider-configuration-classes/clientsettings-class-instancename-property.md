# <a name="clientsettings-class---instancename-property"></a>ClientSettings, classe (propriété InstanceName)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  Obtient le nom de l'instance du client [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
object.InstanceName [= value]  
```  
  
## <a name="parts"></a>Éléments  
 *object*  
 Objet **ClientSettings** qui représente des paramètres sur une instance du client [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .  
  
## <a name="property-valuereturn-value"></a>Valeur de propriété/valeur de retour  
 Valeur de chaîne qui spécifie le nom de l'instance du client [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .
