# <a name="sqlallocstmt-visual-foxpro-odbc-driver"></a>SQLAllocStmt (pilote ODBC Visual FoxPro)
> [!NOTE]  
>  Cette rubrique contient des informations spécifiques au pilote ODBC Visual FoxPro. Pour obtenir des informations générales sur cette fonction, consultez la rubrique appropriée sous [ODBC API Reference](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Prise en charge : Complète  
  
 Conformité d’API ODBC : Niveau principal  
  
 Alloue la mémoire pour un descripteur d’instruction et associe le handle d’instruction à la connexion spécifiée par *pas*. Le Gestionnaire de pilotes transmet cet appel au pilote, qui alloue la mémoire pour le *hstmt* structure.  
  
 Pour plus d’informations, consultez [SQLAllocStmt](../../odbc/reference/syntax/sqlallocstmt-function.md) dans le *de référence du programmeur ODBC*.
