# <a name="odbc-errors"></a>Erreurs ODBC
Lorsqu’une erreur se produit, les pilotes de base de données Microsoft ODBC Desktop retourner le numéro d’erreur natif, la variable SQLSTATE (un code d’erreur ODBC) et un message d’erreur.  
  
 Cette section contient les rubriques suivantes.

-   [Messages d’erreur](../../odbc/microsoft/odbc-jet-error-messages.md)  
  
-   [Erreurs arithmétiques](../../odbc/microsoft/arithmetic-errors.md)
