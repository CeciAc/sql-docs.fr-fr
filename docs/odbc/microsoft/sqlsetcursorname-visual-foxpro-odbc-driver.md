# <a name="sqlsetcursorname-visual-foxpro-odbc-driver"></a>SQLSetCursorName (pilote ODBC Visual FoxPro)
> [!NOTE]  
>  Cette rubrique contient des informations spécifiques au pilote ODBC Visual FoxPro. Pour obtenir des informations générales sur cette fonction, consultez la rubrique appropriée sous [ODBC API Reference](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Prise en charge : Complète  
  
 Conformité d’API ODBC : Niveau principal  
  
 Associe un nom de curseur avec un descripteur d’instruction active, *hstmt*. **SQLSetCursorName** est incluse dans l’API pilote ODBC Visual FoxPro, car il fait partie de la fonctionnalité de base au niveau d’API ODBC ; il ne peut pas être utilisé avec d’autres fonctions d’API, car le pilote ne prend pas en charge les mises à jour positionnées.  
  
 Pour plus d’informations, consultez [SQLSetCursorName](../../odbc/reference/syntax/sqlsetcursorname-function.md) dans le *de référence du programmeur ODBC*.
