# <a name="sqlstate"></a>SQLSTATE
Pour les erreurs qui se produisent dans la source de données, le pilote ODBC mappe l’erreur natif retourné avec le SQLSTATE approprié. Pour les erreurs qui sont détectés par le pilote ou le Gestionnaire de pilotes, le pilote ODBC ou le Gestionnaire de pilotes génère le SQLSTATE approprié.
