# <a name="sqlfreestmt-visual-foxpro-odbc-driver"></a>SQLFreeStmt (pilote ODBC Visual FoxPro)
> [!NOTE]  
>  Cette rubrique contient des informations spécifiques au pilote ODBC Visual FoxPro. Pour obtenir des informations générales sur cette fonction, consultez la rubrique appropriée sous [ODBC API Reference](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Prise en charge : Complète  
  
 Conformité d’API ODBC : Niveau principal  
  
 Arrête le traitement associé à un spécifique *hstmt*, ferme tous les curseurs ouverts, associés à la *hstmt*, ignore les résultats en attente et éventuellement libère toutes les ressources associées à l’instruction handle.  
  
 Pour plus d’informations, consultez [SQLFreeStmt](../../odbc/reference/syntax/sqlfreestmt-function.md) dans le *de référence du programmeur ODBC*.
