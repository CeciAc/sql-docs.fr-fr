# <a name="drop-index-statement"></a>DROP INDEX, instruction
Lorsque le pilote Microsoft Access, dBASE ou Paradox est utilisé, la syntaxe de l’instruction DROP INDEX est « DROP INDEX a b on » où « a » est le nom de l’index et « b » est le nom de la table (DROP pas les INDEX *nom de l’index*).  
  
 Lorsque le pilote Paradox est utilisé, l’instruction DROP INDEX supprime les fichiers d’index secondaire Paradox.  
  
 L’instruction DROP INDEX n’est pas pris en charge pour les pilotes Microsoft Excel ou texte.
