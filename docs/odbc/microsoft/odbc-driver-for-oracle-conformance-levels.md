# <a name="odbc-driver-for-oracle-conformance-levels"></a>Niveaux de conformité du pilote ODBC pour Oracle
> [!IMPORTANT]  
>  Cette fonctionnalité sera supprimée dans une future version de Windows. Évitez d'utiliser cette fonctionnalité dans de nouveaux travaux de développement, et prévoyez de modifier les applications qui utilisent actuellement cette fonctionnalité. Au lieu de cela, utilisez le pilote ODBC fourni par Oracle.  
  
 ODBC définit deux types de normes de conformité pour les pilotes : la conformité des API standard et la conformité de la grammaire SQL standard. Conformité de l’API fait référence aux fonctions prenant en charge un pilote. La mise en conformité SQL fait référence à la grammaire SQL prenant en charge le pilote. Chaque mise en conformité standard se compose des niveaux.  
  
 Cette section fournit des informations de conformité sur le pilote ODBC pour Oracle.
