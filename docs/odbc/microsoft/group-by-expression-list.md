# <a name="group-by-expression-list"></a>GROUP BY, liste d’expressions
GROUP BY la prend en charge une liste d’expressions ainsi que d’un nom de colonne. La liste de sélection d’une instruction SELECT qui contient une clause GROUP BY peut uniquement inclure des expressions de la clause GROUP BY ou définir des fonctions.
