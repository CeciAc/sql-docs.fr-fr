# <a name="convert-function-limitations"></a>CONVERT, fonction - limitations
Résultat d’échecs de conversion de type dans la colonne affectée est définie sur NULL.  
  
 Type de données TIMESTAMP ni DATE peut être convertie en une autre data type (ou lui-même) par la fonction CONVERT.
