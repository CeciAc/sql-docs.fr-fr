# <a name="sqlgetstmtoption-desktop-database-drivers"></a>SQLGetStmtOption (pilotes pour les bases de données de poste de travail)
Les signets retournés par une *fOption* de SQL_GETBOOKMARK sont valides uniquement pendant que la requête est ouverte et sont invalidés lorsque la requête est réémise. Signets persistants ne sont pas pris en charge.
