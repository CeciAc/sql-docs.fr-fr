# <a name="sqlnumparams-visual-foxpro-odbc-driver"></a>SQLNumParams (pilote ODBC Visual FoxPro)
> [!NOTE]  
>  Cette rubrique contient des informations spécifiques au pilote ODBC Visual FoxPro. Pour obtenir des informations générales sur cette fonction, consultez la rubrique appropriée sous [ODBC API Reference](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Prise en charge : complète  
  
 Conformité d’API ODBC : 2 de niveau  
  
 Retourne le nombre de paramètres dans une instruction SQL. Le nombre de paramètres doit égal au nombre de points d’interrogation dans l’instruction SQL transmise au [SQLPrepare](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md).  
  
 Pour plus d’informations sur la grammaire SQL, consultez [pris en charge ODBC SQL grammaire](../../odbc/microsoft/supported-odbc-sql-grammar-visual-foxpro-odbc-driver.md). Pour plus d’informations sur cette fonction, consultez [SQLNumParams](../../odbc/reference/syntax/sqlnumparams-function.md) dans le *de référence du programmeur ODBC*.
