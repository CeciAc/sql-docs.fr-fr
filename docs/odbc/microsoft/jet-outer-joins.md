# <a name="jet-outer-joins"></a>Jet : Jointures externes
Une instruction SELECT peut contenir une liste de clauses OUTER JOIN. JOINTURES externes imbriquées sont prises en charge. Pour plus d’informations sur les jointures externes, consultez le *-Guide du programmeur moteur Microsoft Jet de base de données*.
