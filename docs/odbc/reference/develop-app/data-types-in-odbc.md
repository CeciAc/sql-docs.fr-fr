# <a name="data-types-in-odbc"></a>Types de données dans ODBC
ODBC utilise deux ensembles de types de données : Types de données SQL et les types de données C. Types de données SQL sont utilisés dans la source de données et les types de données C sont utilisés dans le code C dans l’application.
