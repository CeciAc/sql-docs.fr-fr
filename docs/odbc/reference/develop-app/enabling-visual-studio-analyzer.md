# <a name="enabling-visual-studio-analyzer"></a>Activation de Visual Studio Analyzer
> [!IMPORTANT]  
>  Prise en charge pour Visual Studio Analyzer a été supprimée à compter de 8 Windows (Visual Studio Analyzer a été inclus uniquement dans les versions antérieures de Visual Studio.). Pour un autre mécanisme de résolution des problèmes, utilisez le suivi.  
  
 Microsoft Visual Studio Analyzer est un outil d’analyse des application conçu pour fournir un aperçu général de performances d’une application sur tous les niveaux et les systèmes. Il se concentre sur l’interaction entre les composants. Visual Studio Analyzer peut être démarré et arrêté à partir de l’onglet traçage de l’administrateur ODBC.  
  
 Pour afficher les événements qui génère d’ODBC, procédez comme suit :  
  
1.  Accéder à l’administrateur de sources de données ODBC.  
  
2.  Cliquez sur le **suivi** onglet.  
  
3.  Cliquez sur **démarrer le suivi de Visual Studio Analyzer**.  
  
4.  Cliquez sur **OK**.  
  
 Visual Studio Analyzer-génération des événements continue jusqu'à **arrêter Visual Studio Analyzer suivi** est sélectionné.
