# <a name="constructing-sql-statements"></a>Construction d’instructions SQL
Instructions SQL peuvent être construites dans une des trois façons : codées en dur pendant le développement, construites au moment de l’exécution, ou entré directement par l’utilisateur.  
  
 Cette section contient les rubriques suivantes.
