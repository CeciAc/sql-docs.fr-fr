# <a name="visual-studio-analyzer"></a>Visual Studio Analyzer
> [!IMPORTANT]  
>  Prise en charge pour Visual Studio Analyzer a été supprimée à compter de 8 Windows (Visual Studio Analyzer a été inclus uniquement dans les versions antérieures de Visual Studio.). Pour un autre mécanisme de résolution des problèmes, utilisez le suivi.  
  
 Microsoft® Visual Studio™ Analyzer offre une vue d’ensemble de l’application ODBC. Le développeur peut évaluer, analyser et déboguer la structure, les performances et les interactions du point de vue de l’application, plutôt qu’à partir d’un point de vue de composant ou de code. Les informations collectées par Visual Studio Analyzer sont sous la forme d’événements, qui représentent une sorte d’interaction entre les deux composants de l’application. Les événements peuvent être composées d’appels de fonction et retourne à partir de fonctions, telles que les connexions, les requêtes de base de données et les transactions.
