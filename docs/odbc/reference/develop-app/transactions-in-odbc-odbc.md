# <a name="transactions-in-odbc-odbc"></a>Transactions dans ODBC
Transactions dans ODBC sont effectuées au niveau de connexion ; Autrement dit, lorsqu’une application termine une transaction, il valide ou restaure tout le travail effectué par le biais tous les descripteurs d’instruction sur cette connexion.
