# <a name="sqlsetparam-function"></a>SQLSetParam, fonction
**Conformité**  
 Version introduite : Conformité aux normes 1.0 ODBC : Déconseillé  
  
 **Résumé**  
 Dans ODBC 2.0, la fonction ODBC 1.0 **SQLSetParam** a été remplacé par **SQLBindParameter**. Pour plus d’informations, consultez [SQLBindParameter, fonction](../../../odbc/reference/syntax/sqlbindparameter-function.md).  
  
> [!NOTE]  
>  Pour plus d’informations sur quelles le Gestionnaire de pilotes mappe cette fonction lorsqu’un ODBC 2. *x* application fonctionne avec un ODBC 3. *x* pilote, consultez [mappage de fonctions déconseillées](../../../odbc/reference/appendixes/mapping-deprecated-functions.md).  
  
## <a name="remarks"></a>Notes  
 Consultez [informations sur ODBC 64 bits](../../../odbc/reference/odbc-64-bit-information.md), si votre application s’exécutera sur un système d’exploitation 64 bits.
