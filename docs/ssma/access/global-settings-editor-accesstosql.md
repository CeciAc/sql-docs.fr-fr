# <a name="global-settings-editor-accesstosql"></a>Paramètres globaux (éditeur) (AccessToSQL)
Utilisez la page de l’éditeur de la **paramètres globaux** boîte de dialogue pour spécifier les paramètres pour les fenêtres d’éditeur de code.  
  
Pour accéder aux paramètres de l’éditeur, sur le **outils** menu, sélectionnez **paramètres globaux**, cliquez sur **GUI** en bas du volet gauche, puis sélectionnez **éditeur**.  
  
## <a name="options"></a>Options  
**Afficher les numéros de ligne dans les éditeurs**  
Pour afficher les numéros de ligne dans les fenêtres de l’éditeur de code, sélectionnez **True**.  
  
Si une fenêtre d’éditeur de code est activée lorsque vous modifiez ce paramètre, vous devrez peut-être sélectionner à nouveau l’objet actuel à mettre à jour l’affichage.
