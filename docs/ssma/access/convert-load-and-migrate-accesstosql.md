# <a name="convert-load-and-migrate-accesstosql"></a>Convertir, charger et migrer (AccessToSQL)
Le **convertir, charger et migrer** boîte de dialogue signale l’état de migration lorsque vous utilisez le bouton Convertir, de chargement et de migrer ou l’Assistant de Migration.  
  
Une fois la migration terminée, vous pouvez afficher d’information, avertissement et messages d’erreur en cliquant sur le lien dans chaque section. Le lien montre comment de nombreux messages d’erreur, avertissement et d’information sont disponibles, telles que « 0 erreurs, 4 avertissements. »  
  
Le **migrer des données pour les objets sélectionnés** section de la page contient également un **rapport** bouton. Vous pouvez cliquer sur ce bouton pour afficher un rapport de migration de données.  
  
## <a name="options"></a>Options  
**Rapport**  
Affiche le rapport de conversion ou la migration dynamique.
