# <a name="installing-ssma--for-oracle-oracletosql"></a>Installation de SSMA pour Oracle (OracleToSQL)
[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Migration Assistant (SSMA) pour Oracle se compose d’une application cliente qui permet d’effectuer une migration d’Oracle vers [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] et base de données SQL Azure. Il contient également un pack d’extension qui prend en charge la migration des données et l’utilisation de fonctions de système Oracle dans vos bases de données migrées.  
  
Vous installez l’application cliente sur l’ordinateur à partir duquel vous allez effectuer les étapes de migration. Vous devez installer les fichiers du pack extension sur l’ordinateur hébergeant les bases de données migrées. Cet ordinateur doit être en cours d’exécution [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
## <a name="upgrading-ssma-for-oracle"></a>La mise à niveau de SSMA pour Oracle  
Si vous souhaitez mettre à niveau vers une version ultérieure de SSMA pour Oracle, vous devez d’abord désinstaller le client et le pack d’extension de serveur, puis installez la version plus récente.
