# <a name="mssql-cli-command-line-query-tool-for-sql-server"></a>outil de requête de ligne de commande MSSQL-CLI pour SQL Server
[!INCLUDE[appliesto-ss-asdb-asdw-xxx-md](../includes/appliesto-ss-asdb-asdw-xxx-md.md)]

MSSQL-CLI est un outil de ligne de commande interactif permettant d’interroger SQL Server, de l’installer sur Windows, macOS ou Linux.

## <a name="install-mssql-cli"></a>Installer MSSQL-CLI

Pour obtenir des instructions d’installation détaillées, consultez le [Guide d’installation](https://github.com/dbcli/mssql-cli/blob/master/doc/installation_guide.md)de ou, si vous connaissez PIP, installez-le en exécutant la commande suivante:

```$ pip install mssql-cli```
