# <a name="calculationcurrentpass-mdx"></a>CalculationCurrentPass (MDX)


  Retourne le test de calcul actuel d'un cube pour le contexte de requête spécifié.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
CalculationCurrentPass()  
```  
  
## <a name="remarks"></a>Notes  
 Le **CalculationCurrentPass** fonction retourne l’index de base zéro de la passe de calcul pour le contexte de requête actuel. Avec une résolution automatique de la récursivité, cette fonction a peu utilisation pratique.
