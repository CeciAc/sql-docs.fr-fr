# <a name="schema-compare-extension-preview"></a>Extension de comparaison de schéma (préversion)
L’extension de comparaison de schémas fournit une expérience facile à utiliser pour comparer des fichiers et des bases de données .dacpac et appliquer les modifications de la source à la cible.

Cette expérience est actuellement dans sa préversion initiale. Signalez les problèmes et les demandes de fonctionnalités [ici.](https://github.com/microsoft/azuredatastudio/issues)

## <a name="install-the-schema-compare-extension"></a>Installer l’extension de comparaison de schémas

1. Pour ouvrir le gestionnaire d’extensions et accéder aux extensions disponibles, sélectionnez l’icône d’extensions ou sélectionnez **Extensions** dans le menu **Affichage**.
2. Sélectionnez une extension disponible pour afficher ses détails.
1. Sélectionnez l’extension de votre choix (comparaison de schémas) et **installez-la**.

## <a name="how-do-i-start-a-schema-comparison"></a>Comment démarrer une comparaison de schémas ?
* Le point d’entrée principal pour la comparaison de schémas est de cliquer avec le bouton droit sur une base de données dans l’explorateur d’objets, puis de cliquer sur **Comparaison de schémas**.
* L’utilisateur peut également lancer la boîte de dialogue de comparaison de schémas à partir de la palette de commandes (Ctrl+Maj+P) en recherchant **Comparaison de schémas**

## <a name="why-would-i-use-the-schema-compare"></a>Pourquoi utiliser la comparaison de schémas ?
La comparaison de schémas a été créée pour ajouter la possibilité de comparer les schémas des fichiers .dacpac et des bases de données et d’appliquer les modifications.
