# <a name="schema-compare-extension-preview"></a>Extension de comparaison de schéma (version préliminaire)
L’extension de comparaison de schémas fournit une expérience facile à utiliser pour comparer les bases de données et fichiers .dacpac et appliquer les modifications à partir de la source à la cible.

Cette expérience est actuellement dans sa version préliminaire. Signaler des problèmes et demandes de fonctionnalités [ici.](https://github.com/microsoft/azuredatastudio/issues)

## <a name="install-the-schema-compare-extension"></a>Installer l’extension de comparaison de schémas

1. Pour ouvrir le Gestionnaire d’extensions et accéder aux extensions disponibles, sélectionnez l’icône des extensions ou **Extensions** dans le **vue** menu.
2. Sélectionner une extension disponible pour afficher ses détails.
1. Sélectionnez l’extension que vous souhaitez (comparaison de schémas) et **installer** il.

## <a name="how-do-i-start-a-schema-comparison"></a>Comment commencer une comparaison de schéma ?
* Le point d’entrée principal pour la comparaison de schémas est avec le bouton droit à une base de données dans l’Explorateur d’objets, cliquez sur **comparaison de schémas**.
* L’utilisateur peut également lancer la boîte de dialogue de comparaison de schéma à partir de la palette de commandes (Ctrl + Maj + P) en recherchant **comparaison de schémas**

## <a name="why-would-i-use-the-schema-compare"></a>Pourquoi utiliser la comparaison de schémas ?
Comparaison de schéma a été créée pour ajouter la possibilité de comparer les schémas à partir de bases de données et fichiers .dacpac et appliquer les modifications.
