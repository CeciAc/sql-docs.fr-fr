# <a name="monitor-the-appliance-with-system-views---analytics-platform-system"></a>Surveiller l’appliance avec les vues système - Analytique Platform System
Cet article répertorie les vues système que vous pouvez utiliser pour la surveillance de SQL Server PDW.  
  
## <a name="to-monitor-the-appliance-by-using-system-views"></a>Pour surveiller l’Appliance à l’aide de vues système  
SQL Server PDW inclut les vues système complète qui vous permettent d’obtenir des informations détaillées sur l’appliance, état et les performances. Ce tableau fournit des liens vers des vues système qui peuvent être utilisées pour chaque fonctionnalité de surveillance.
