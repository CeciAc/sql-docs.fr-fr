# <a name="configurationsetting-property---virtualdirectoryreportserver"></a>Propriété ConfigurationSetting - VirtualDirectoryReportServer
  Retourne le répertoire virtuel de l'application de serveur de rapports. En lecture seule.  
  
## <a name="syntax"></a>Syntaxe  
  
```vb  
Public Dim VirtualDirectoryReportServer As String  
```  
  
```csharp  
public string VirtualDirectoryReportServer;  
```  
  
## <a name="property-values"></a>Valeurs de la propriété  
 **Chaîne** qui contient le répertoire virtuel.
